# -有关每日电力负荷量的时间序列预测
  # -各文件说明
    commom.py:时间序列预测的数据预处理阶段，主要功能是读取和清洗时间序列数据,为后续的特征工程和模型训练准备干净、有序的数据。
    
    log.py:这是一个功能完善的Python日志管理类，专门为电力负荷预测等长期运行项目设计。该类提供了专业级的日志记录解决方案，能够自动创建日志目录、设置多级别日志输出，并确保线程安全。核心功能包括智能的日志级别映射（从debug到crit五个级别）、灵活的格式定制，以及自动化的文件管理——无需手动创建目录即可直接使用。通过简单的配置，开发者就能快速获得一个支持中文、具备时间戳、可追溯的日志系统，极大提升了项目的可维护性和调试效率。特别适合需要长期运行、多模块协作的数据分析项目，为电力负荷预测等关键业务提供可靠的运行记录和错误追踪能力。
    
    train.csv：作为模型训练的训练集，其拥有16752个数据，每个数据规范格式为:'年/月/日 时:分'(year/month/day hour:minute),数据范围为:2013/9/2 0:00至2015/7/31 23:00。
    test.csv:作为模型训练的测试集，其拥有16752个数据，每个数据规范格式为:'年/月/日 时:分'(year/month/day hour:minute),数据范围为:2013/9/2 0:00至2015/7/31 23:00。
